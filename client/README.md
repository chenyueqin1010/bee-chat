Recommended PNPM
----

#### Usage

```bash
#install all
pnpm install

#install one
pnpm add <pkg>
pnpm add -D <pkg>
pnpm add <pkg>@next
pnpm add <pkg>@3.0.0	

#run
pnpm <cmd>
```
